===================================================
Horizon-Drift - Documentation
===================================================

Welcome to Horizon-Drift!

Horizon-Drift is a custom Minecraft Launcher designed to be fast, beautiful, and easy to use.

Features:
- Supports Vanilla, Fabric, and Forge installations.
- Connects to Ely.by, Microsoft, and Offline accounts.
- Modrinth integration for downloading modpacks.
- Custom RAM allocation and JVM arguments.
- Glassmorphism UI and smooth animations.

Troubleshooting:
- If the launcher fails to start, ensure you have the latest graphics drivers installed.
- If Minecraft fails to launch, check the "Logs" tab for more information.
- If you experience lag, try allocating more RAM in the "Settings" tab or the "Installations" tab.

Uninstalling:
To uninstall Horizon-Drift, open your Windows Control Panel, go to "Add or Remove Programs", find "Horizon-Drift", and click Uninstall.

Enjoy your game!
